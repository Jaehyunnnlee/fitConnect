# 프로젝트 목적 및 기능
SpringBoot를 이용한 헬스 커뮤니티 구현

-Spring security를 이용한 로그인/회원가입/마이페이지(회원정보 수정,삭제) 기능

-게시판/게시글 기능

-댓글/답글 기능

-나의 운동 루틴 기록 및 공유 기능
